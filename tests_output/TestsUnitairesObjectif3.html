3.3.1.1.1.1. Cas normal
3.3.1.1.1.1.1. Obtention du chemin le plus court dans un graph
But du test : Creation d'un graph de 10 noeuds, composes d'arcs. On cherche le chemin le plus court entre le noeud 1 et le noeud 10.
Resultat attendu : Resultat obtenu :
Le reponse doit etre la suivante : 1, 4, 5, 7, 2, 10 avec un cout de 12.
Dijkstra
total cost : 12
path : 1 4 5 7 2 10
Conclusion : Nous avons bien recupere le chemin le plus court.

Resultat attendu : Resultat obtenu :
Le server doit nous indiquer qu'il n'existe pas. --------------------------------
## Error : Can't find a path ##
--------------------------------
